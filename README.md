# BLITZKRIEG - Tech Fest Website

The official website for Blitzkrieg, the annual tech fest of Keshav Mahavidyalaya, University of Delhi.

## 🚀 Live Site
Visit live at: https://blitzkriegkmv.github.io

## 🛠️ Tech Stack
- **React 18** - UI Library
- **Vite** - Build Tool
- **React Router** - Client-side Routing
- **Swiper** - Carousel/Slider
- **CSS Modules** - Scoped Styling

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📁 Project Structure

```
src/
├── components/     # Reusable components (Navbar, Footer)
├── pages/          # Page components (Home, Events, Team, Sponsors)
├── styles/         # Global styles
├── App.jsx         # Main App component with routing
└── main.jsx        # Entry point
public/
├── media/          # Images and media files
└── fonts/          # Custom fonts
```

## 🎨 Features
- Responsive design for all devices
- Interactive event modals
- Team carousel with Swiper
- Modern glassmorphism navbar
- Dark theme throughout
